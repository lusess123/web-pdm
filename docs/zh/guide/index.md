# 清晰呈现复杂的数据关系

`web-pdm` 是一个轻量的 React ER 图组件。它可以把模型元数据转换成交互式画布，并提供模型导航、关系连线、自动布局、缩放和图片导出等能力。

## 为什么需要 ER 工作台？

ER 图为产品、领域专家和工程团队提供同一套系统视图。无论是评审数据库结构、讨论领域边界、估算功能工作量，还是解释模块之间的数据流向，都可以围绕一张图展开。

## 为嵌入业务系统而设计

这个包是一个组件，而不是托管式建模服务。模型数据、事件回调、国际化文案和外围界面都由你的应用掌控。你可以直接使用默认的轻量控件，也可以通过组件适配接口替换它们。

## 渲染方式

当前图层基于 G6 5.1.1，并通过独立的运行时适配层接入。G6 负责 Canvas 渲染、布局、端口和图交互，React 负责模型导航和操作控件。这个边界可以保持组件公开接口稳定，同时隔离 G6 的异步生命周期。
