# web-pdm 历史资料

本目录保留 web-pdm 早期版本的目标、界面截图、路线图和技术致谢。它们是项目演进的一部分，不会因为技术栈升级而删除。当前安装方式、公开接口和运行命令请同时参考仓库根目录的 `README.md`、`README.zh-CN.md` 以及 `docs/` 文档站源码。

## 介绍

web-pdm 最初被定义为一个用 G6 制作的 ER 图工具，最终目标是做成在线版 PowerDesigner。

![web-pdm 早期 ER 图界面](./erd.jpeg)

## 缘起

[《ER 图的设计与实现》](https://www.yuque.com/antv/g6-blog/nbaywp)记录了项目最初的设计思路。

PowerDesigner 的建模能力和中文元信息支持影响了很多数据库设计工作，但早期版本主要面向 Windows。web-pdm 希望把这类能力带到浏览器中，使领域专家、产品人员和工程师可以围绕同一张模型图协作。

## 在线体验

早期文档曾提供以下两个独立端口的 Demo：

- [定制版 Demo](http://zyking.xyz:5080/demo/ '定制版 Demo')
- [基本版 Demo](http://zyking.xyz:5002/ '基本版 Demo')

这两个历史地址可能已经停止服务。当前在线体验统一使用 [erd.zyking.xyz](https://erd.zyking.xyz/)。

## 历史启动方式

```bash
npm i
npm run watch
```

> **历史实现说明**：`npm run watch` 属于早期构建链路。当前仓库使用 pnpm 和 Rspress，开发命令为 `pnpm dev`。保留旧命令是为了帮助维护旧项目和阅读历史提交。

## 当前架构说明

当前版本已经迁移到 G6 5，并以 `web-pdm` 和 `web-pdm-core` 两个包分别提供可直接使用的轻量界面与核心能力。默认界面不再依赖 Ant Design，文档站也不再使用 dumi/father；这些变化不会改变早期版本对“在线 PowerDesigner”的产品目标。
