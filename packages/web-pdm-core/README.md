# web-pdm-core

The G6 5 graph runtime, state model and public TypeScript types used by
[`web-pdm`](https://www.npmjs.com/package/web-pdm).

Most applications should install `web-pdm`, which provides the ready-to-use
lightweight controls and icons. Use this core package when you need to supply a
fully customized component and icon layer.

```bash
pnpm add web-pdm-core
```

- [Documentation](https://erd.zyking.xyz/config/)
- [Migration notes](https://erd.zyking.xyz/guide/migration/)
- [GitHub](https://github.com/lusess123/web-pdm)

Version 3 uses G6 5 while retaining the documented web-pdm data, event and
component customization interfaces.
